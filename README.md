# RaccoonLabs — Official Website

![RaccoonLabs](https://img.shields.io/badge/Built%20by-RaccoonLabs-blueviolet)

The official RaccoonLabs website and landing page, hosted at [cyberraccoonteam.github.io](https://cyberraccoonteam.github.io).

## What It Is
This is the source for the RaccoonLabs web presence — showcasing projects, API products, and documentation for the CyberRaccoonTeam ecosystem. Built with clean, static HTML/CSS for speed and simplicity.

## Tech Stack
- HTML5, CSS3, JavaScript (vanilla)
- GitHub Pages

## Quick Start
```bash
git clone https://github.com/CyberRaccoonTeam/cyberraccoonteam.github.io.git
cd cyberraccoonteam.github.io
# Open index.html in your browser
```
